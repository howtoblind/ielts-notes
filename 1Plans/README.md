###制定一个灵活的(flexible)的计划

人很难在一开始就给自己制定一个长期适用的计划，所以在复习计划制定的时候，务必考虑计划灵活性，在计划中给自己备有充足的**调整时间**，用来在某个特殊的日子，比如肚子疼、亲属拜访、好友邀请春游……的时候，调整时间分配，利用之前预留的备用的时间把进度赶回来。
